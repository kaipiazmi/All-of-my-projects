--------------Cook App--------------

Adalah aplikasi yang dibuat melihat bahan dan resep-resep untuk masakan yang enak dan aplikasi ini tidak akan dikembangkan,
karna agar mengingat aplikasi yang pertama kali saya buat, saya sadar masih banyak kekurangan, akan saya perbaiki di project berikutnya, 
dibuat untuk menyelesaikan project akhir bootcamp flutter developer batch 39 Sanbercode

Halaman
1.SplashScreen
2.Register & Login dengan Autentifikasi firebase
3.halaman Home (masih tersedia 4 jenis resep masakan)
4.halaman detail bahan dan cara pembuatan menggunakan rest api +(postman)
5.halaman About me yaitu tentang developer(pembuat aplikasi yaitu saya sendiri)
6.halaman tentang aplikasi...
 
Screenshot tampilan aplikasi ada di folder UI>ScreenshotApp
halaman detail yang menggunakan rest api makanan ada di folder makanan
